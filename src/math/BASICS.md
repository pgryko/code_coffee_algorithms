# Sum of finite arithmetic series

```math
S_n = \sum_{n=1}^{n} a+ (n-1)d

S_n = a + (a + d) + (a + 2d) + ... +  (a + (n-2)d) + (a + (n-1)d)
S_n = (a + (n-1)d) + (a + (n-2)d) + ... + (a + 2d) + (a + d) + a

2S_n = n(2a + (n-1)d)
S_n = \frac{n}{2}(2a + (n-1)d)

```
